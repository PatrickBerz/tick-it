Hello, welcome to TickIt!

TickIt was designed/developed by Jayde Holbrook, Van Hudson, Patrick Berzins, Ben Lukins, and Nathaniel Smith as part of the Computer Science Senior Design course at the University of Alabama in Huntsville (CS 499).

If you are reading this file, you are already halfway to installing TickIt! Please ensure that Node.js is installed on your machine. Once Node.js in installed, if you are using Windows, you can simply use the included Batch Script file to download any needed dependencies and launch the web app.

Linux and Macintosh users can use the included Shell Script file. Extern must be installed for the Shell Script to work.
